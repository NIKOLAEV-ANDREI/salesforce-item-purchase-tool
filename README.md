# Item Purchase Tool

Salesforce application for creating purchases from an Account record.

## Development environment

- Salesforce Developer Org
- Salesforce CLI
- Visual Studio Code with Salesforce Extension Pack

## Project structure

- `force-app/main/default` — Salesforce metadata: Apex, LWC, objects, flows, permissions.
- `config` — project configuration.
- `manifest` — metadata manifests for retrieve and deploy operations.

## Default org

The local alias of the development organization is `item-purchase-dev`.
